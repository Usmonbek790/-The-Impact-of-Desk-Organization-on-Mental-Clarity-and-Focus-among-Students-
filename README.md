# Desk Organization and Mental Clarity Survey

## Overview
This is a Streamlit web application that evaluates how workspace organization affects mental clarity and productivity.

## Files
- `streamlit_app.py` - main application
- `questions.json` - external questions file
- `survey_results.json` - saved results
- `requirements.txt` - dependencies

## Run locally
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
